# Explainable Multimodal Medical Diagnosis Using Chest X-ray Images and Clinical Text

This repository contains a production-ready PyTorch implementation of a multimodal deep learning system designed for medical diagnosis. It combines visual features from Chest X-ray images with semantic features from radiology reports to predict a multi-label set of diseases. 

Crucially, this system incorporates **Explainable AI (XAI)** techniques to ensure clinical interpretability.

## 🚀 Features
* **Multimodal Fusion Engine**: Concatenates image embeddings extracted via a pre-trained **DenseNet121** and text embeddings generated by **BERT**.
* **XAI Capabilities**:
  * **Grad-CAM**: Generates heatmaps on Chest X-rays highlighting the visual regions responsible for a specific disease prediction.
  * **Self-Attention**: Extracts BERT attention weights to highlight critical keywords in the clinical text.
* **Production Codebase**: Modular object-oriented architecture (`data_loader`, `model_image`, `model_text`, `fusion_model`, `train`, `evaluate`, `explainability`).
* **Multi-Label Classification**: Robustly handles sparse datasets with hundreds of potential diagnostic labels.

## 🧠 Model Architecture
1. **Image Modality**: Takes normalized 224x224 RGB images and passes them through a pre-trained `densenet121` without the final classifier layer, yielding a `1024` dimension feature vector.
2. **Text Modality**: Takes tokenized report text (findings + impression) of length `128` and passes them through `bert-base-uncased`, yielding a `768` dimension `[CLS]` token embedding.
3. **Fusion**: Vectors are concatenated `[1024 + 768 = 1792]` and passed through a Multi-Layer Perceptron (MLP) with Batch Normalization and Dropout to produce the final multi-label logits.

## 📊 Evaluation
Run the evaluation script to calculate the micro F1-score and generate ROC curves for the top 10 most frequent diseases.
```bash
python -m src.evaluate
```

## 🔦 Explainability
The system justifies its predictions:
1. It targets the final convolutional layer of DenseNet to generate spatial heatmaps.
2. It targets the final attention layer of BERT to visualize word-importance scores.

To test this out, run the demo notebook or import functions from `src.explainability`.

## 🛠️ Usage
1. Clone the repository.
2. Install requirements: `pip install -r requirements.txt`.
3. Preprocess data: `python src/data_preprocessing.py`.
4. Train the model: `python src/train.py --batch_size 32 --epochs 10`.
5. Evaluate the model: `python -m src.evaluate`.
